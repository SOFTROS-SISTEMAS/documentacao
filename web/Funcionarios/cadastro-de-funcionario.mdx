title: "Cadastro de funcionário"
---

Esta segunda maneira de cadastrar um funcionário no sistema **Agulhão** tem como primeiro passo aplicar uma **ação** ao cliente. A ação a ser aplicada a este cliente é a de "**adicionar como funcionário**". Após o cliente ter sido adicionado como **funcionário**, a sequência do cadastro deste novo funcionário é continuada no menu **Pessoal-funcionario**, seguindo o passo a passo do vídeo abaixo.
