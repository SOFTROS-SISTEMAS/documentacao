# Integração com Pix Sicoob

Para integrar o sistema Agulhão com o Pix do Sicoob, siga os passos abaixo.

## 1. Criar uma conta no Portal de Desenvolvedores Sicoob

Primeiro, acesse o Portal de Desenvolvedores do Sicoob no link abaixo e faça o cadastro ou login:

https://developers.sicoob.com.br/portal/login

## 2. Cadastro de um novo aplicativo

Com sua conta logada, crie um novo aplicativo para produção seguindo o tutorial oficial:

https://developers.sicoob.com.br/portal/documentacao?slugItem=apis-do-sicoob&slugSubItem=aplicativos-de-producao

- Na criação do aplicativo, selecione "Sim" para indicar que o certificado será de uma empresa parceira.
- Escolha "SOFTROS SISTEMAS" como a empresa parceira.

## 3. Liberação e autorização do aplicativo na conta bancária

Após criar o aplicativo em produção, será necessário liberar o acesso no aplicativo do banco. Para isso, siga este tutorial passo a passo:

https://developers.sicoob.com.br/portal/documentacao?slugItem=seguranca&slugSubItem=autorizacao

## 4. Obtenção do Client ID

Com o aplicativo criado e liberado, você terá acesso ao **Client ID**.

## 5. Configuração do Agulhão

Após obter o Client ID, entre em contato com o suporte do Agulhão para realizar a configuração da conta e do webhook, finalizando a integração.
