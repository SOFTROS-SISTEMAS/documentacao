O sistema Agulhão Web oferece um menu exclusivo para cadastro e gestão de fornecedores.

Para cadastrar um fornecedor, siga estes passos:

1. Acesse o menu **Fornecedores** e clique em **Fornecedores**.
2. Na página de cadastro, clique no botão verde **Inserir**.
3. Preencha todas as informações do fornecedor.
4. Defina o plano de contas do fornecedor e outras informações necessárias.
5. Clique no botão azul **Confirmar** para finalizar o cadastro.

Definir o plano de contas é essencial para relacionar o fornecedor às notas XML baixadas.
