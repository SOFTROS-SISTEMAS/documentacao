## **Botões de Navegação**

Os botões de navegação são amplamente utilizados durante o uso do aplicativo Agulhão, seja para iniciar um pedido, consultar inadimplentes, retornar ao menu inicial ou entrar em contato com o suporte.

Na **engrenagem** também tem o botão de sincronização neste botão você pode estar sincronizando todas as informações selecionada previamente

Você tem total liberdade para sincronizar pedidos, produtos e clientes, escolhendo o período desejado para a sincronização das informações.

Se o botão **“não sincronizar os pedidos automaticamente”** estiver selecionado, o aplicativo Agulhão não sincronizará os pedidos diretamente com o sistema web, e os pedidos aparecerão com o status de desincronizados.
