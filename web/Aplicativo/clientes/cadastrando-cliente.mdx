---
title: "Cadastrando Clientes"
description: "Agora veremos como é possível cadastrar um cliente dentro do nosso App."
---

Este botão abre a ficha cadastral para inclusão de novos clientes, com todos os campos necessários para o cadastro de pessoa física ou jurídica.

Para cadastrar um novo cliente, basta clicar no botão **“\+”**.

<img
  src="/images/Untitled(80).png"
  alt="Untitled (80).png"
  title="Untitled (80).png"
  style={{ width:"53%" }}
/>

Ao clicar no botão **“\+”**, será exibida uma nova tela (como mostrado abaixo), onde você poderá preencher todas as informações do cliente.

<img
  src="/images/Untitled(81).png"
  alt="Untitled (81).png"
  title="Untitled (81).png"
  style={{ width:"54%" }}
/>

A ficha cadastral possui identificação automatizada do cliente. Ao clicar no botão **“Adicionar Imagem”**, você pode tirar uma foto, seja do próprio cliente ou da fachada da loja.

O cadastro de cliente também permite adicionar a localização. Clicando no botão de **localização**, você poderá registrar o endereço da loja e/ou do cliente.

Após preencher todas as informações, clique no botão **“Concluir”**. O novo cliente será cadastrado automaticamente no sistema Agulhão Web.
