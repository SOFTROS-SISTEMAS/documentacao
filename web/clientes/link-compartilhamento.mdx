title: "Link de Compartilhamento"
---

O link de compartilhamento \*\* da venda, foi desenvolvido para otimizar os processos na troca de informações sobre o pedido(**Pedidos de Saída**) e a Venda(**Frente de Caixa**), entre a empresa e o cliente.

Esta solução otimiza as trocas de informações com: visualizar todos os produtos do pedido com valores, envio de boletos para o cliente, acompanhamento do status do pedido do cliente, o envio de NFe e XML para o cliente.

Com o link de rastreio, o cliente pode acompanhar qual status o pedido se encontra: recebido pela fábrica, aprovando, preparando ou financeiro, entre outros. No caso da Loja no varejo o cliente acompanha outros status como: condicional, aberto e fechado.

O seu cliente também consegue acompanhar os boletos que estão pendentes e os que já foram quitados.

**Para gerar o Link de Compartilhamento.**

Esta solução otimizada as troca de informações com: visualizar todos os produtos do pedido com valores, envio de boletos para o cliente, o cliente acompanha os status do pedido, o envio de NFe e XML para o cliente.

Com o link de rastreio, o cliente pode acompanhar qual status o pedido se encontra, recebido pela fábrica, aprovando, preparando ou financeiro, entre outros. No caso da Loja no varejo o cliente acompanha outros status como: condicional, aberto e fechado.

1 - Acesse o Pedido de saída ou Frente de Caixa.

2 - Localize o pedido/venda.

3 - Clique nas ações do pedido/venda, no canto direito.

4 - Selecione a opção**Gerar Link Compartilhamento** .

5 - Escaneie pelo QR code ou envie por WhatsApp para o cliente ou copie o link e envie para o cliente de outra maneira. Assista ao vídeo **Como Gerar o link de compartilhamento?** Clicando no link abaixo.
