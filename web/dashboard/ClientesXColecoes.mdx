---
title: "Clientes x Coleções. "
description: "Dashboard que analisa vendas por coleção para clientes, com acesso a histórico e dados financeiros."
---

Esta página de pesquisa de informações, destinada para facções, tem como objetivo oferecer uma análise detalhada das vendas por coleção para cada cliente.

O Dashboard Cliente x Coleções permite analisar as compras de cada cliente referentes às coleções filtradas por você. É possível selecionar quantas coleções desejar, e o sistema Agulhão realizará a busca das vendas correspondentes para cada cliente.

Além disso, o Cliente x Coleções possibilita acessar o histórico completo de cada cliente clicando nos três risquinhos, onde você encontrará informações sobre duplicatas, renegociações e outros dados relevantes.
